3. a) What do mean by virtual function ? Explain with example.

  • Virtual function is a member function that is declared within a base class and redefined by a derived class.
  • To create virtual function, procede the function's declaration in the base class with the keyword "virtual". 
        
    Example of virtual function:


    #include <iostream>
    using namespace std;

    // Base class definition
    class Base {
    public:
    // Virtual function to print a message
    virtual void print() {
        cout << "Base class is invoked" << endl;
    }
    };

    // Derived class inheriting from Base
    class Derived : public Base {
    public:
    // Overridden function to print a different message
    void print() override {
        cout << "Derived class is invoked" << endl;
    }
    };

    int main() {
    /* Creating a pointer to the Base class, 
    pointing to an object of the Derived class.*/
    Base* base = new Derived();

    // Calling the print function through the base class pointer
    base->print(); // prints "Derived class is invoked"

    // Deleting the dynamically allocated object to avoid memory leaks
    delete base;

    return 0;
    }


    Output:
    Derived class is invoked


    /*In this example, the print() function is a virtual function
    in the Base class. This means that it can be overridden in derived
    classes. In the Derived class, we override the print() function 
    to print a different message.

    When we create a pointer to a Base object and assign it to a 
    Derived object, we can still call the print() function on the base 
    pointer. However, because the print() function is virtual, the 
    overridden print() function in the Derived class will be called.*/

5. b) What do you mean by dynamic binding ? How is it useful in OOP ?

  • Dynamic binding, also known as late binding or runtime binding, is a feature of object-oriented programming.
  • It allows a program to determine the correct method to call at runtime, rather than at compile-time.
    Dynamic binding is useful in OOP because:
    • It allows us to ignore the type differences by providing us with the flexibility in choosing which type of function we need at that instant of runtime.
    • It allows the same method or operator to have different behaviors depending on the type or state of the objects that invoke or interact with it.
    • It allows runtime lookup of virtual functions.
    • It allows us to have virtual functions in parent classes which can be overriden in derived classes.
    Dynamic binding is achieved through the use of virtual functions.

6. a) What is 'this' pointer? How will you use 'this' pointer in C++? Explain with an example.

  • In C++ 'this' is a keyword that refers to the current instance of the class.
    • Usage of 'this' pointer:
    • It can be used to pass current object as a parameter to another method.
    • It can be used to refer current class instance variable.
        
    Example:

    #include <iostream>
    using namespace std;

    // Define the Employee class
    class Employee
    {
    public:
        // Member variables
        int id;
        string name;
        float salary;

        // Constructor to initialize the Employee object
        Employee(int id, string name, float salary)
        {
            /* 'this' pointer is used to distinguish 
            between class members and constructor parameters. */
            this->id = id;
            this->name = name;
            this->salary = salary;
        }

        // Member function to display employee information
        void display()
        {
            // 'this' pointer is implicitly available within member functions
            cout << "Employee id: " << this->id << endl;
            cout << "Employee name: " << this->name << endl;
            cout << "Employee salary: " << this->salary << endl;
        }
    };

    int main()
    {
        // Creating an Employee object using the constructor
        Employee e1 = Employee(105, "Anikesh", 30000);

        // Displaying employee information.
        e1.display();

        return 0;
    }

    Output:

    Employe id: 105
    Employe name: Anikesh
    Employe salary: 30000

6. b) What are the various access specifier used in C++? Explain with an example.

  • Access Specifier define how the mambers of a class can be accessed.
    • There are three types of acces specifier in C++.
    • Public: Mambers are accessible from outside of the class.
    • Private: Mambers cannot be accessed or view from outside of the class.
    • Protected: Mambers cannot be accessible from out side of the class, however they can be accessed in inherited classes.
        
    Example of access specifiers.

    #include <iostream>
    #include <string>
    using namespace std;

    class Person {
    public:
        // Public members
        int publicId;

        // Public member function
        void displayPublicInfo() {
            cout << "Public ID: " << publicId << endl;
        }

    private:
        // Private members
        string privateName;

        // Private member function
        void displayPrivateInfo() {
            cout << "Private Name: " << privateName << endl;
        }

    protected:
        // Protected members
        int protectedAge;

        // Protected member function
        void displayProtectedInfo() {
            cout << "Protected Age: " << protectedAge << endl;
        }

    public:
        // Public member function to access private member
        void setPrivateName(string name) {
            privateName = name;
        }
    };

    int main() {
        // Create an object of the Person class
        Person person;

        // Accessing public members
        person.publicId = 123;
        person.displayPublicInfo();

        // Accessing protected members
        person.protectedAge = 25;
        person.displayProtectedInfo();

        // Accessing private members indirectly through public member functions
        person.setPrivateName("John");
        // person.privateName = "John"; // This would result in an error
        // person.displayPrivateInfo(); // This would result in an error

        return 0;
    }

    Output:

    Public ID: 123
    Protected Age: 25

7. a) How will you define constants in C++? How will you access them Explain with suitable example.

  • Constants can be defined globally or locally.
  • Constants are declared using the 'const' keyword.
  • Constants are variables that cannot be changed once they have been initialized.
  • Constants can be of any data type, including integers, floating-point numbers, and strings.
  • Global constants are accessible to all functions in the program.
  • Local constants are only accessible to the function in which they are defined.
        
    Example of constants:

    #include <iostream>
    #include <string>  
    using namespace std;
            
    // Global constants
    const double PI = 3.14159;
    const int MAX_VALUE = 100;
            
    // Local constants
    int main() 
    {
        const int MIN_VALUE = 0;
        const string GREETING = "Hello, world!";
            
        // Use global constants
        cout << "The value of PI is " << PI << endl;
            
        // Use local constants
        cout << "The value of MIN_VALUE is " << MIN_VALUE << endl;
        cout << GREETING << endl;
            
        return 0;

    The value of PI is 3.14159
    The value of MIN_VALUE is 0
